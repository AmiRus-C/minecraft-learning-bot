## 🎯 Сценарии Использования

### Сценарий 1: Простое исследование
```javascript
applyPreset(bot, 'friendlyExplorer');
// Бот будет исследовать мир и собирать ресурсы
```

### Сценарий 2: Серьёзная работа
```javascript
applyPreset(bot, 'seriousCommander');
// Бот будет работать методично и осторожно
```

### Сценарий 3: Игра в двоём
```javascript
await startCoopGameMode(bot);
coop.setTeamGoal('bed_wars');
// Бот будет помогать реальному игроку
```

### Сценарий 4: Полная персонализация
```javascript
bot.setPersonality({
  personality_name: 'MyBot',
  speech_style: 'casual',
  traits: { curiosity: 0.9, friendliness: 0.8 }
});
// Бот с полностью настроенной личностью
```
