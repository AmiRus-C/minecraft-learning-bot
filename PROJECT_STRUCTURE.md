# 📋 Проектные файлы Minecraft Learning Bot

## 📂 Структура проекта

```
MinecraftBot/
│
├── 📄 package.json                    # Зависимости Node.js
├── 📄 README.md                       # Основная документация
├── 📄 INSTALLATION.md                 # Инструкция по установке
├── 📄 GUIDE.md                        # Полное руководство
├── 📄 QUICKSTART.js                   # Быстрый старт и примеры
│
├── 📁 config/
│   ├── 📄 botConfig.js               # Основная конфигурация
│   └── 📄 personalityPresets.js       # Предустановки личности (10 вариантов)
│
├── 📁 src/
│   ├── 📄 bot.js                     # Главный файл бота (все системы)
│   │
│   ├── 📁 systems/
│   │   ├── 📄 PersonalitySystem.js   # Личность, эмоции, память
│   │   ├── 📄 ChatSystem.js          # Общение в чате (4 стиля)
│   │   ├── 📄 WorldInteractionSystem.js # Взаимодействие с миром
│   │   └── 📄 CooperativeGameMode.js # Режим игры в двоём
│   │
│   ├── 📁 ai/
│   │   └── 📄 LearningSystem.js      # Q-Learning алгоритм обучения
│   │

│       ├── 📄 Logger.js              # Логирование
│       └── 📄 BotCLI.js              # CLI интерфейс
│
├── 📁 data/
│   ├── 📄 personality.db             # БД личности (автоматическая)
│   └── 📄 learning.db                # БД обучения (автоматическая)
│
└── 📁 logs/
    └── 📄 bot.log                    # Логи выполнения (автоматический)
```

## 📚 Описание файлов

### Корневые файлы

| Файл | Описание |
|------|---------|
| `package.json` | Зависимости: mineflayer, nedb, chalk |
| `README.md` | Основная документация и особенности |
| `INSTALLATION.md` | Подробная инструкция установки и настройки |
| `GUIDE.md` | Полное руководство с примерами |
| `QUICKSTART.js` | Быстрый старт и готовые стартапы |

### 📁 config/

**botConfig.js**
- Параметры подключения к серверу
- Параметры поведения (скорость, дальность взаимодействия)
- Параметры памяти и обучения
- Параметры логирования

**personalityPresets.js**
- 10 готовых личностей (Explorer, Commander, Assistant и т.д.)
- Функция `applyPreset()` для применения
- Функция `listPresets()` для просмотра

### 📁 src/

**bot.js** (600+ строк)
- Главный класс MinecraftBot
- Инициализация всех систем
- Обработчики событий
- Главный цикл AI
- 9 различных действий бота

**src/systems/**

**PersonalitySystem.js**
- Управление личностью и эмоциями
- Система памяти событий
- Отслеживание предпочтений
- Сохранение в БД

**ChatSystem.js**
- 4 стиля речи (friendly, formal, casual, mysterious)
- 5 категорий сообщений
- Генерация динамических ответов
- Вероятностные сообщения

**WorldInteractionSystem.js**
- Движение к целям
- Добыча блоков
- Размещение блоков
- Атака мобов
- Сбор предметов
- Прыжки и навигация

**CooperativeGameMode.js**
- Следование за игроком
- Защита от врагов
- Совместная добыча
- Скоординированная атака
- Bed Wars стратегия
- Система помощи

**src/ai/**

**LearningSystem.js**
- Q-Learning алгоритм
- ε-greedy策略 (epsilon-greedy)
- Сохранение знаний в БД
- Статистика обучения
- Адаптивное исследование

**src/utils/**

**Logger.js**
- Логирование в консоль
- Цветной вывод (chalk)
- Запись в файлы
- Разные уровни логирования

**BotCLI.js**
- Интерактивный интерфейс команд
- Управление личностью
- Просмотр статуса
- Отладочная информация

## 🎯 Ключевые особенности по файлам

### PersonalitySystem.js
- ✅ Полностью настраиваемая личность
- ✅ 5 черт характера
- ✅ 5 эмоций
- ✅ Система памяти (макс 1000 событий)
- ✅ Отслеживание предпочтений
- ✅ Сохранение в NeDB

### LearningSystem.js
- ✅ Q-Learning с параметрами обучения
- ✅ ε-greedy исследование (20% случайных действий)
- ✅ Адаптивное снижение исследования
- ✅ Сохранение знаний между сеансами
- ✅ Макс 10 000 опытов в памяти

- ✅ Динамическая генерация ответов

- ✅ Вероятностные сообщения (спам-контроль)

## 💡 Примеры быстрого старта

### 1️⃣ Дружелюбный исследователь
```javascript
applyPreset(bot, 'friendlyExplorer');
```

### 2️⃣ Серьёзный командир
```javascript
applyPreset(bot, 'seriousCommander');
```

### 3️⃣ Игра в двоём (Bed Wars)
```javascript
await startCoopGameMode(bot);
bot.setTeamGoal('bed_wars');
```

### 4️⃣ Кастомная личность
```javascript
bot.setPersonality({
  personality_name: 'Alex',
  speech_style: 'friendly',
  traits: { friendliness: 0.9, curiosity: 0.8 }
});
```
