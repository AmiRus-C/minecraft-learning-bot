# 📖 Полное Руководство по Установке

## ✅ Требования

- **Node.js** версия 16 или выше ([скачать](https://nodejs.org))
- **npm** (поставляется с Node.js)
- **Minecraft сервер** версия 1.16.5
  - Оригинальный сервер Minecraft
  - Или Spigot/Paper сервер на 1.16.5

## 🚀 Пошаговая установка

### Шаг 1: Проверка Node.js

Откройте PowerShell и проверьте версию Node.js:

```powershell
node --version
npm --version
```

Должны быть версии 16+ и 7+.

### Шаг 2: Установка зависимостей

```powershell
cd c:\Users\Lenovo\Desktop\FLClient\MinecraftBot
npm install
```

Ждите завершения установки. Это может занять 2-5 минут.

### Шаг 3: Подготовка Minecraft сервера

#### Вариант А: Локальный сервер (Рекомендуется)

1. **Скачайте** server.jar для версии 1.16.5 с официального сайта Minecraft
2. **Создайте папку** для сервера (например, `C:\MinecraftServer`)
3. **Поместите** server.jar в эту папку
4. **Создайте файл** `run.bat` с содержимым:

```batch
@echo off
java -Xmx1024M -Xms1024M -jar server.jar nogui
pause
```

5. **Запустите** run.bat (первый запуск создаст файлы конфигурации)
6. **Отредактируйте** server.properties:
   - `enable-rcon=false` (если не нужен удалённый доступ)
   - `server-port=25565` (по умолчанию)
   - `level-type=flat` (опционально, для удобства тестирования)

7. **Запустите сервер снова**

#### Вариант Б: Публичный сервер

Если вы используете публичный сервер, отредактируйте `config/botConfig.js`:

```javascript
connection: {
  host: 'your-server-ip',  // IP вашего сервера
  port: 25565,             // Порт сервера
  username: 'MinecraftAI',  // Имя бота (должно быть свободно)
  version: '1.16.5'
}
```

### Шаг 4: Запуск бота

```powershell
# Простой запуск
npm start

# Или с автоперезагрузкой при изменении файлов
npm run dev
```

Вы должны увидеть логи подключения и сообщение о появлении бота в мире.

## ⚙️ Конфигурация

### Основные параметры

Отредактируйте `config/botConfig.js`:

```javascript
// Подключение
connection: {
  host: 'localhost',
  port: 25565,
  username: 'MinecraftAI',
  version: '1.16.5'
},

// Скорость движения
behavior: {
  walkSpeed: 4.3,
  sprintSpeed: 5.6
},

// Параметры обучения
learning: {
  explorationRate: 0.1,      // 10% исследования
  learningRate: 0.7,         // Скорость обучения
  discountFactor: 0.9        // Ценность будущего опыта
}
```

### Установка личности

В файле `src/bot.js` найдите функцию `main()` и раскомментируйте:

```javascript
async function main() {
  const bot = new MinecraftBot();

  // РАСКОММЕНТИРУЙТЕ ЭТО:
  bot.setPersonality({
    personality_name: 'Alex',
    personality_description: 'Опытный исследователь',
    speech_style: 'friendly',
    traits: {
      friendliness: 0.8,
      curiosity: 0.9,
      confidence: 0.7,
      cautiousness: 0.4,
      creativity: 0.6
    }
  });

  await bot.initialize();
  // ...
}
```

## 🆘 Решение проблем

### Проблема: "Bot не подключается"

**Решение:**
1. Проверьте, запущен ли Minecraft сервер
2. Проверьте хост и порт в `config/botConfig.js`
3. Убедитесь, что используете версию 1.16.5
4. Проверьте файрвол

### Проблема: "Cannot find module 'mineflayer'"

**Решение:**
```powershell
# Переустановите зависимости
npm install
```

### Проблема: "Bot подключается но не появляется"

**Решение:**
1. Проверьте логи в `logs/bot.log`
2. Убедитесь, что имя бота (`username`) не совпадает с другими игроками
3. Перезагрузите сервер

### Проблема: "Bot застревает на месте"

**Решение:**
1. Это нормально для базовой навигации - она простая
2. Попробуйте перезагрузить бота
3. Используйте плоский мир (flat) для тестирования

### Проблема: "npm install не работает"

**Решение:**
```powershell
# Очистите кэш npm
npm cache clean --force

# Переустановите
npm install
```
