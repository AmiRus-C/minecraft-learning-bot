## 💡 Примеры Использования

### Простой боток
```javascript
npm start
bot> set-preset friendlyExplorer
// Боть будет исследовать мир
```

### Серьёзный работник
```javascript
npm start
bot> set-preset seriousCommander
bot> goals set mining_quota
// Боть будет добывать ресурсы
```

### Игра в двоём
```javascript
// Отредактируйте src/bot.js
await startCoopGameMode(bot);
// Боть будет помогать игроку
```

### Сохранение и загрузка
```
bot> profiles save my_explorer
bot> profiles load my_explorer
```

### Запись стратегий
```
bot> record start my_strategy
// ... боть что-то делает ...
bot> record stop
bot> record play my_strategy
```

## 📞 Как Начать?

### Шаг 1: Скачать и установить
```bash
cd MinecraftBot
npm install
```

### Шаг 2: Запустить Minecraft сервер
```bash
# Локально или удалённо (версия 1.16.5)
java -Xmx1024M -Xms1024M -jar server.jar nogui
```

### Шаг 3: Запустить бота
```bash
npm start
```

### Шаг 4: Управлять ботом
```
bot> help
bot> set-preset friendlyExplorer
bot> status
```
